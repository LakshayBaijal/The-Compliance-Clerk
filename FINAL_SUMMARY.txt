HOW TO RUN
═════════════════════════════════════════════════════════════════════════════

Prerequisites:
  • Python 3.11+ installed
  • Virtual environment created and activated
  • GROQ_API_KEY set in .env file

Execute Pipeline:
  $ python -m src.main sample.pdf --output results.xlsx
  $ python -m src.main ./pdfs/ --with-reports
  $ python -m src.main ./pdfs/ --use-llm --recursive --with-reports

Run Tests:
  $ pytest -q              # Quick run
  $ pytest -v              # Verbose output
  $ pytest --cov=src       # With coverage

Output Files:
  • compliance_results_TIMESTAMP.xlsx (Excel workbook)
  • batch_report_TIMESTAMP.txt (Processing statistics)
  • performance_report_TIMESTAMP.txt (Timing metrics)
  • performance_data_TIMESTAMP.json (Machine-readable data)
  • logs/audit.db (SQLite audit trail)
  • logs/compliance_clerk.log (Application logs)
